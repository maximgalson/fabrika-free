# 5 агентных воркфлоу

Готовые рецепты для параллельных агентов в Claude Code. Копируй промпт, вставляй, запускай.

---

## 1. Контент на неделю за 2 минуты

Попроси Claude Code запустить 3 агента одновременно:

```
Запусти 3 агента параллельно:

Агент 1: Придумай 7 тем для контента на неделю в моей нише. Для каждой — хук.

Агент 2: Напиши 3 поста для Threads по первым трём темам. С хуком, структурой и CTA.

Агент 3: Напиши 3 сценария Reels по темам 4-6. С тайм-кодами, 30-60 сек каждый.
```

Результат: 7 тем + 3 поста + 3 сценария. За 2 минуты.

---

## 2. Анализ конкурентов

```
Запусти 3 агента параллельно:

Агент 1: Найди 5 популярных тем в нише [твоя ниша], которые сейчас набирают охваты. Для каждой — почему работает.

Агент 2: Придумай как я могу адаптировать каждую из этих тем под свой стиль и аудиторию.

Агент 3: Напиши 5 хуков — по одному на каждую тему. Провокационные, цепляющие.
```

---

## 3. Генерация идей + хуки + посты

```
Запусти 3 агента параллельно:

Агент 1: 10 идей для контента в моей нише. Короткий список.

Агент 2: 10 хуков для постов — провокация, число+обещание, личная история. По одному на каждую идею из агента 1... ладно, просто 10 хуков в моей нише.

Агент 3: Напиши 3 поста для Threads на любые 3 темы из моей ниши. Готовые к публикации.
```

---

## 4. Исследование аудитории

```
Запусти 3 агента параллельно:

Агент 1: 10 болей моей целевой аудитории. Конкретные, не абстрактные.

Агент 2: 10 желаний моей целевой аудитории. Чего они хотят на самом деле.

Агент 3: 5 возражений, которые мешают моей аудитории купить мой продукт. И как их снять контентом.
```

---

## 5. Аудит контент-стратегии

```
Запусти 2 агента параллельно:

Агент 1: Проанализируй мой профиль эксперта. Что сильно, что можно усилить. Какие темы я не раскрываю, а стоило бы.

Агент 2: Составь контент-стратегию на месяц: 4 недели, каждая с темой и 5 идеями. Чередуй типы контента: экспертный, личный, продающий, вовлекающий.
```

---

## Как запустить агентов

В Claude Code просто напиши промпт выше. Claude сам запустит агентов параллельно — ты увидишь как несколько задач выполняются одновременно.

Для лучшего результата — убедись что у тебя есть профиль эксперта (CLAUDE-FABRIKA.md). Агенты читают его для персонализации.
