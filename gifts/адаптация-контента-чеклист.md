# Чеклист адаптации контента

Одна идея — контент для разных платформ. Не копипаст, а переосмысление под формат.

---

## Платформы и их правила

### Threads
- Макс 500 символов на ветку
- Хук в первой строке — решает всё
- Без ссылок (убивают охват)
- Без хештегов
- CTA: вопрос или "сохрани"
- Тон: острый, плотный, каждое слово на месте

### Telegram
- 800-1500 символов
- Жирная первая строка
- Короткие абзацы (2-3 предложения)
- Можно раскрыть тему глубже — контекст, пример, кейс
- Без хештегов
- CTA: вопрос или "сохрани"
- Тон: личный, как сообщение другу

### Reels / Shorts / TikTok
- 30-60 секунд
- Первые 3 секунды = 80% успеха
- Устная речь, не текст
- Один ролик = одна мысль
- CTA: подписка, комментарий или кодовое слово
- Тон: энергичный, прямой

### YouTube
- 8-15 минут (длинный формат)
- Заголовок + превью решают клик
- Структура: хук → проблема → решение → CTA
- Можно глубоко раскрыть тему
- CTA: подписка + ссылка в описании

### Instagram карусель
- 8-12 слайдов
- Первый слайд = обложка (должен цеплять)
- Один слайд = одна мысль (макс 3 строки)
- Последний слайд = CTA
- Тон: авторитетный, структурированный

---

## Что менять при адаптации

| Элемент | Threads | Telegram | Reels |
|---------|---------|----------|-------|
| Длина | 500 симв | 800-1500 | 30-60 сек |
| Хук | текст, 1 строка | жирный текст | визуал + голос |
| Глубина | поверхностно | глубоко | средне |
| Тон | острый | личный | энергичный |
| CTA | вопрос | вопрос/сохрани | подписка/коммент |
| Ссылки | нет | можно | в описании |

## Что НЕ менять

- Ключевую мысль — одна и та же везде
- Голос эксперта — тон адаптируется, личность нет
- Аудиторию — пишешь для тех же людей

## Быстрый способ

Скопируй в Claude Code:
```
/fabrika:demo-adapt [твоя тема]
```
