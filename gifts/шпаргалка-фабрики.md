## Команды курса

| Команда | Что делает |
|---------|-----------|
| `/fabrika:start` | Обзор курса |
| `/fabrika:urok-1` | Профиль эксперта |
| `/fabrika:urok-2` | Первый пост (промпт vs скилл) |
| `/fabrika:urok-3` | Одна идея → три платформы |
| `/fabrika:urok-4` | Параллельные агенты |
| `/fabrika:urok-5` | MCP, Notion, лендинг |
| `/fabrika:urok-6` | Полная картина |

## Демо-скиллы (работают всегда)

| Команда | Что делает |
|---------|-----------|
| `/fabrika:demo-threads` | Пост для Threads за минуту |
| `/fabrika:demo-reels` | Сценарий Reels за минуту |
| `/fabrika:demo-adapt` | Одна идея → Threads + Telegram + Reels |

## Быстрые промпты

```
Придумай 7 тем на неделю с хуками
```

```
Напиши пост для Threads на тему: [тема]
```

```
Адаптируй эту идею под Threads + Telegram + Reels: [идея]
```

```
Запусти 3 агента: идеи + хуки + посты
```

```
Напиши сценарий Reels на тему: [тема]
```

## Профиль эксперта

Твой профиль лежит в `CLAUDE-FABRIKA.md` или `CLAUDE.md`. Claude Code читает его при каждом разговоре. Чтобы обновить — просто отредактируй файл.
