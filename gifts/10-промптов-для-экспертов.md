# 10 промптов для экспертов

Эти промпты работают в 10 раз лучше, если у тебя есть профиль эксперта (CLAUDE-FABRIKA.md или CLAUDE.md). Попробуй с профилем и без — почувствуешь разницу.

Копируй и вставляй в Claude Code.

---

## Контент

### 1. Идеи на неделю
```
Придумай 7 идей для контента на эту неделю. Для каждой — тема + хук (первая строка поста). Учитывай мою нишу и аудиторию из профиля.
```

### 2. Пост из одной мысли
```
Я хочу написать пост на тему: [тема]. Напиши пост для Threads — с хуком, структурой и CTA. В моём стиле.
```

### 3. Разбор чужого поста
```
Вот пост конкурента: [вставить текст]. Разбери: что работает, что нет. Предложи как я бы написал на ту же тему — в своём стиле.
```

## Стратегия

### 4. Боли аудитории
```
Назови 10 болей моей целевой аудитории. Для каждой — тема поста, которая эту боль затрагивает.
```

### 5. Контент-стратегия на месяц
```
Составь контент-стратегию на месяц: 4 недели, каждая с темой недели и 5 идеями постов. Учитывай мою нишу и платформы.
```

### 6. Позиционирование
```
На основе моего профиля — сформулируй моё позиционирование в одном предложении. Кто я, для кого, чем отличаюсь.
```

## Продажи

### 7. Продающий пост
```
Напиши пост, который продаёт [мой продукт/услугу] через историю. Не в лоб — через боль аудитории и решение. В моём стиле.
```

### 8. Лид-магнит
```
Придумай 5 вариантов бесплатного лид-магнита для моей аудитории. Для каждого — название + что внутри + почему захотят скачать.
```

## Продуктивность

### 9. Переработка контента
```
Вот мой пост: [вставить текст]. Адаптируй его для Telegram (длиннее, глубже) и для Reels (сценарий 30-60 сек с тайм-кодами).
```

### 10. Заголовки и хуки
```
Напиши 10 хуков (первых строк) для постов в моей нише. Используй формулы: провокация, число+обещание, личная история. Каждый — цепляющий.
```
