# Шаблон Notion-базы для контента

Готовая база данных Notion для управления контентом. Скопируй к себе по ссылке:

**https://galson.notion.site/fabrika-free-course?v=46d177fd502d4a4bb76663113892e235**

(Открой → "Duplicate" в правом верхнем углу → база скопируется в твой Notion)

---

## Поля базы данных

| Поле | Тип | Значения |
|------|-----|----------|
| Название | Title | Заголовок/тема поста |
| Платформа | Select | Threads, Telegram, Reels, YouTube, Carousel |
| Статус | Select | Идея, Черновик, Готов, Опубликован |
| Дата публикации | Date | Когда публикуется |
| Тип контента | Select | Экспертный, Личный, Продающий, Вовлекающий |
| Хук | Text | Первая строка поста |
| CTA | Select | Вопрос, Сохрани, Подписка, Ссылка |
| Результат | Select | Хит, Норма, Слабо |

## Статусы и workflow

```
Идея → Черновик → Готов → Опубликован
                    ↓
              (отметить результат)
```

1. **Идея** — тема записана, ещё не написана
2. **Черновик** — пост написан (через Claude Code), нужна проверка
3. **Готов** — проверен, можно публиковать
4. **Опубликован** — опубликован, ждём результат

## Вьюхи (фильтры)

### На сегодня
Фильтр: Статус = "Готов" + Дата = сегодня

### На эту неделю
Фильтр: Дата = эта неделя, сортировка по дате

### Идеи (банк тем)
Фильтр: Статус = "Идея", сортировка по дате создания

### Что сработало
Фильтр: Результат = "Хит", сортировка по дате

## Как пользоваться

1. Каждую неделю генерируй 7 идей (через Claude Code или `/fabrika:demo-threads`)
2. Записывай в Notion со статусом "Идея"
3. Пиши посты — меняй статус на "Черновик"
4. Проверяй — меняй на "Готов"
5. Публикуй — меняй на "Опубликован"
6. Через день отмечай результат: Хит / Норма / Слабо
7. Раз в месяц смотри что сработало — делай больше такого

## В Фабрике Контента

В полной Фабрике Claude Code подключается к Notion через MCP и сам записывает посты в базу. Ты только проверяешь и публикуешь.
