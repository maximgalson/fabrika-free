# Контент-план на неделю — шаблон

Используй этот шаблон каждую неделю. Заполни темы сам или попроси Claude Code:

```
Придумай 7 тем для контента на эту неделю. Для каждой — хук. Учитывай мой профиль.
```

## Типы контента (чередуй)

- **Экспертный** — советы, разборы, инструкции (40%)
- **Личный** — истории, выводы, закулисье (30%)
- **Продающий** — кейсы, результаты, оффер (20%)
- **Вовлекающий** — вопросы, опросы, дискуссии (10%)

## Как пользоваться

1. В начале недели заполни темы (сам или через Claude)
2. Каждый день бери тему и запускай `/fabrika:demo-threads` или `/fabrika:demo-adapt`
3. Проверь, опубликуй
4. В конце недели — отметь что зашло, что нет

## Быстрая генерация

Скопируй в Claude Code:
```
У меня контент-план на неделю. Тема понедельника: [тема]. Напиши пост для Threads в моём стиле.
```
