# Фабрика Контента — Бесплатный курс для Claude Code

Интерактивный мини-курс, который за 30 минут покажет, как AI создаёт контент в твоём стиле для любой платформы.

## Шаг 1: Установи Claude Code

Claude Code — это AI-ассистент, который работает прямо в терминале твоего компьютера. Фабрика Контента работает внутри него.

**Если Claude Code уже установлен — переходи к Шагу 2.**

Если нет — установи:

1. Открой **Терминал** (на Mac: Finder → Программы → Утилиты → Терминал)
2. Скопируй и вставь эту команду:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

3. Следуй инструкциям на экране
4. После установки набери `claude` — если появится приветствие, всё работает

Подробная инструкция: [claude.ai/download](https://claude.ai/download)

## Шаг 2: Установи бесплатный курс

В том же терминале скопируй и вставь:

```bash
curl -sSL https://raw.githubusercontent.com/maximgalson/fabrika-free/main/install.sh | bash
```

Для Windows (PowerShell):
```powershell
irm https://raw.githubusercontent.com/maximgalson/fabrika-free/main/install.ps1 | iex
```

## Шаг 3: Запусти курс

1. Набери `claude` (если ещё не запущен)
2. Набери `/fabrika:start`
3. Следуй инструкциям на экране

## Что внутри

**6 уроков:**

| Команда | Что произойдёт |
|---------|---------------|
| `/fabrika:start` | Обзор курса |
| `/fabrika:urok-1` | AI узнает тебя — настройка под твой стиль и нишу |
| `/fabrika:urok-2` | Первый пост за 60 секунд — увидишь разницу |
| `/fabrika:urok-3` | Одна идея — контент для трёх платформ |
| `/fabrika:urok-4` | Твоя AI-команда — несколько Claude одновременно |
| `/fabrika:urok-5` | Инструменты — MCP, Notion, лендинг |
| `/fabrika:urok-6` | Как всё связано в систему |

**3 бонус-скилла (работают всегда):**

| Команда | Что делает |
|---------|-----------|
| `/fabrika:demo-threads` | Пост для Threads за минуту |
| `/fabrika:demo-reels` | Сценарий Reels за минуту |
| `/fabrika:demo-adapt` | Одна идея — три платформы |

**6 подарков** — промпты, шаблоны, воркфлоу, Notion-шаблон. Открываются по мере прохождения.

## Требования

Для работы курса нужна **CLI-версия Claude Code** — это приложение, которое работает в терминале (командной строке) твоего компьютера. Не путать с claude.ai в браузере или десктоп-приложением Claude — это другое.

**Claude Code CLI** — это именно терминальный инструмент, в котором ты пишешь команды текстом, а Claude выполняет их: создаёт файлы, пишет контент, запускает агентов.

### Что нужно:

1. **Компьютер** с macOS, Linux или Windows (через WSL)
2. **Подписка Claude** (Pro, Max или Team) — без неё Claude Code не работает
3. **Claude Code CLI** — установленный в терминале

### Как проверить, установлен ли Claude Code:

Открой терминал и набери:
```bash
claude --version
```
Если видишь номер версии — всё ок. Если ошибка "command not found" — нужно установить.

### Как установить Claude Code CLI:

1. Открой **Терминал**:
   - **Mac**: Finder → Программы → Утилиты → Терминал (или Spotlight: `Cmd+Space`, набери "Terminal")
   - **Windows**: Установи [WSL](https://learn.microsoft.com/ru-ru/windows/wsl/install), затем открой Ubuntu из меню Пуск
   - **Linux**: `Ctrl+Alt+T`

2. Вставь команду:
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

3. Следуй инструкциям на экране (авторизация через браузер)

4. Проверь: набери `claude` — если появится приветствие, всё работает

Подробнее: [claude.ai/download](https://claude.ai/download)
