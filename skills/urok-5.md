## Шаг 1: MCP и Notion

Скажи:

**MCP — это как USB-порт для AI.**

Ты подключаешь сервис — и Claude Code может с ним работать. Читать данные, создавать записи, управлять.

Например: подключаешь Notion — и Claude Code может создавать страницы, заполнять базы данных, организовывать контент. Не через браузер — прямо из терминала.

**Вот как это выглядит в реальности.**

В полной Фабрике Контента Notion используется как место, куда складывается весь контент. Посты, идеи, статусы — всё в одной базе. Claude Code генерирует пост и сам записывает его в Notion с нужным статусом и тегами.

Покажи пользователю пример структуры Notion-базы для контента:

```
╔═══════════════════════════════════════════════════════════╗
║                                                           ║
║   ПРИМЕР: Notion-база контента                            ║
║                                                           ║
║   Поля:                                                   ║
║   - Название поста (Title)                                ║
║   - Платформа (Threads / Telegram / Reels / YouTube)      ║
║   - Статус (Черновик / Готов / Опубликован)                ║
║   - Дата публикации                                       ║
║   - Тема / Рубрика                                        ║
║   - Текст поста (внутри страницы)                         ║
║                                                           ║
║   Workflow:                                                ║
║   Claude создаёт пост → записывает в Notion →             ║
║   ты проверяешь → меняешь статус → публикуешь             ║
║                                                           ║
╚═══════════════════════════════════════════════════════════╝
```

Скажи:

**Так работает Фабрика.** Claude Code подключается к Notion через MCP и ведёт всю базу контента за тебя. Пишет посты, редактирует, контролирует их наличие. Ты просто даёшь обратную связь и он учится писать лучше.

Подключение MCP — это настройка один раз. Дальше — работает автоматически.

**В Фабрике Контента ты получишь готовые Notion-шаблоны под разные типы контента — копируешь себе и подключаешь.**

## Шаг 2: Лендинг для лид-магнита

Скажи:

**Контент — это не самоцель. Контент ведёт куда-то.**

Ты пишешь посты → люди читают → и что дальше?

Им нужно место, куда прийти. Страница, где они оставят контакт или получат что-то полезное.

**Это лендинг. И Claude Code может его собрать.**

Прямо сейчас мы соберём простой лендинг для лид-магнита — бесплатного полезного материала, который ты даёшь аудитории в обмен на контакт.

**Что ты мог бы дать аудитории бесплатно?** Чек-лист, гайд, шаблон, мини-курс — что-то полезное из твоей ниши.

Например:
- Коуч → "5 вопросов для самодиагностики"
- Маркетолог → "Чек-лист запуска рекламы"
- Психолог → "Гайд: как понять что пора к специалисту"

**Назови свой лид-магнит.** Что бы ты дал аудитории бесплатно?

Жди ответа.

После ответа скажи:

**Отлично. Лид-магнит: "[ответ пользователя]".**

Сейчас я соберу для тебя лендинг. Одна HTML-страница — с заголовком, описанием, формой и дизайном.

Появится запрос на создание файла — подтверди.

Прочитай профиль пользователя. Создай файл `landing.html` в текущей директории. Лендинг должен:

- Быть одной HTML-страницей с встроенным CSS (без внешних зависимостей)
- Адаптивный (мобильный + десктоп)
- Цветовая схема: тёмный фон, жёлтые акценты (стиль Content Factory)
- Содержать:
  1. Заголовок — цепляющий, на основе лид-магнита пользователя
  2. Подзаголовок — для кого это и что получишь
  3. 3-4 буллета — что внутри лид-магнита
  4. Форма (имя + email/telegram) — с кнопкой "Получить бесплатно"
  5. Блок об авторе — из профиля пользователя
  6. Футер — с ссылками на соцсети
- Форма НЕ рабочая (демо) — в action поставить `#`
- Чистый, профессиональный дизайн

После создания файла открой его в браузере:

```bash
open landing.html
```

Скажи:

**Готово. Открой браузер — твой лендинг уже там.**

Это рабочая HTML-страница. Можешь залить на любой хостинг — и у тебя есть место, куда вести аудиторию из контента.

**Контент → Лид-магнит → Лендинг → Контакт → Продажа.** Вот зачем нужен контент.

А если хочешь что-то изменить — просто скажи. "Сделай заголовок крупнее" или "поменяй цвет кнопки". Claude Code обновит файл и перезагрузит страницу.

Дай пользователю возможность попросить 1-2 правки. Если просит — внеси и открой обновлённый файл.
