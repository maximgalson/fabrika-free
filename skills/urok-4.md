## Шаг 1: Запуск трёх агентов

Скажи:

**Запускаю трёх агентов одновременно.**

Каждый получит свою задачу — все три связаны с твоей нишей.

Сейчас появятся запросы на разрешение — подтверди. Это агенты начинают работу.

Прочитай профиль пользователя. Затем запусти 3 агента ПАРАЛЛЕЛЬНО (в одном сообщении, используя Agent tool 3 раза):

**Агент 1 — Идеи контента:**
Промпт: "Ты — контент-стратег. Прочитай профиль эксперта [путь к CLAUDE-FABRIKA.md или CLAUDE.md]. На основе его ниши и аудитории придумай 5 идей для контента на эту неделю. Для каждой идеи: название + 1 предложение почему эта тема зацепит аудиторию. Формат: нумерованный список. Без вводных слов, сразу список."

**Агент 2 — Анализ ниши:**
Промпт: "Ты — аналитик контента. Прочитай профиль эксперта [путь к CLAUDE-FABRIKA.md или CLAUDE.md]. Найди 3 популярные темы в нише этого эксперта, которые сейчас набирают охваты в соцсетях. Для каждой: тема + почему она работает + как эксперт может адаптировать её под себя. Без вводных слов, сразу анализ."

**Агент 3 — Хуки:**
Промпт: "Ты — копирайтер-хукописец. Прочитай профиль эксперта [путь к CLAUDE-FABRIKA.md или CLAUDE.md]. Напиши 10 хуков (первых строк) для постов в нише этого эксперта. Хуки должны цеплять, провоцировать, вызывать желание читать дальше. Используй формулы: провокация, число+обещание, личная история. Формат: нумерованный список. Без вводных слов, сразу хуки."

После запуска скажи:

**Три агента запущены. Работают параллельно.**

- Агент 1: придумывает 5 идей контента на неделю
- Агент 2: анализирует популярные темы в твоей нише
- Агент 3: пишет 10 хуков для постов

Подожди — результаты придут одновременно.
