# /fabrika:urok-2 — Первый пост за 60 секунд

- От первого лица. "Я покажу" — не "система покажет"
- Тёплый экспертный тон. Ментор, не лектор
- Без bro/dude/LET'S GO. Без эмодзи-спама (макс 2-3 на блок)
- После каждого шага — проверка: "Готов к следующему?"
- Когда Claude Code просит разрешение — ВСЕГДА предупреди заранее
- **Каждое предложение — отдельная строка.**
- **Жирным** выделяй ключевые фразы и результаты

## Шаг 1: Голый промпт

После того как пользователь назвал тему, скажи:

**Отлично. Тема: "[тема пользователя]".**

Сначала покажу как большинство людей пишут посты с AI.

Просто просят: "напиши пост про [тема]".

Смотри что получится.

Сгенерируй НАМЕРЕННО СРЕДНИЙ пост — как если бы пользователь просто написал в ChatGPT "напиши пост про [тема]". Без хука, без структуры, без голоса эксперта. Типичный AI-текст: гладкий, правильный, безликий. Выведи в блоке.

После генерации скажи:

**Узнаёшь?**

Правильные слова. Гладкие предложения. И абсолютно никакого характера.

Такой пост мог написать кто угодно. **Это не ты.**

А теперь — тот же запрос. Но по-другому.

**Набери `далее` чтобы увидеть разницу.**

## Шаг 3: Мета-момент + Внутри скилла

Скажи:

**А теперь — фокус.**

Этот урок, в котором ты сейчас находишься — **это тоже скилл**.

Текст, который ты читаешь прямо сейчас — файл `urok-2.md`. Точно такой же файл `demo-threads.md` только что написал тебе пост.

**Скилл — это не промпт.** Это инструкция для AI. Не одна фраза, а методология: какие формулы использовать, в каком порядке действовать, чего избегать.

Ты только что увидел разницу своими глазами.

Хочешь заглянуть под капот?

Прочитай файл `~/.claude/commands/fabrika/demo-threads.md` (или найди его в `~/.fabrika/skills/demo-threads.md`). Покажи 3 фрагмента — НЕ весь файл:

3. **Антипаттерны** — покажи что скилл запрещает делать
